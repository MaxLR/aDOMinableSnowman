Try adding list items to the list! Grab the unordered list with the command x = $l('.test-ul')
You can then add a new list item with x.append('<li>{Your Text}</li>')
You can empty the list by typing x.empty()

You can set attributes of DOM elements by typing x.attr('{Attribute}', '{Key}')
You can add classes to DOM elements by typing x.addClass('{Class Name}')
You can also remove classes to DOM elements by typing x.removeClass('{ClassName}')
You can add event listeners to DOM elements by using the method x.on('{Event Listener}', {Callback function})
An sample event listener is: x.on('click', () => console.log("Hey! Thanks for clicking!")). Try this one out, then click on the list above!
